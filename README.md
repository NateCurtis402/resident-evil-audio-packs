# 🧟 Resident Evil Audio Packs for Decky Loader

A collection of **Resident Evil themed menu music and UI sound packs** for the **Audio Loader plugin** used with Decky Loader on Steam Deck or Steam Deck UI.

These packs replace the Steam menu ambience with iconic **Resident Evil save room themes, and menu sounds**, bringing that safe haven atmosphere to your Steam interface.

Perfect for Resident Evil fans who want their Steam UI to feel like a **safe room between missions.** 

---

# 🎵 Included Packs

This repository contains multiple audio packs inspired by different Resident Evil games.

Examples include:

- Resident Evil (2002) Save Room Theme
- Resident Evil 2 (1998) Save Room Theme
- Resident Evil Code Veronica X Save Room Theme
- Resident Evil 4 Save Room Theme
- Resident Evil 4 Remake Save Room Theme
- Resident Evil 7 Save Room Theme
- Resident Evil UI Sound Effects
  

More packs may be added in the future. (Just LMK)

# 🎮 Requirements

You must have the following installed:

- **Decky Loader**
- **Audio Loader Plugin**

Audio Loader can be installed directly through the Decky Loader plugin store.

---

# 📦 Installation

1. Navigate to your Decky Loader sounds folder:
~/homebrew/sounds/


2. Clone this repository:
git clone https://github.com/NateCurtis402/resident-evil-audio-packs.git


3. Ensure the pack goes into the correct folder:
~/homebrew/sounds/


5. Open **Steam → Decky Loader → Audio Loader**

6. Select your Resident Evil music pack.

---

# 🎧 What This Does

These packs replace the **Steam Deck UI background music and interface sounds** with audio from the Resident Evil series.

Examples include:

- Save room themes as menu ambience
- Menu navigation sounds
- UI sound effects inspired by Resident Evil inventory systems
